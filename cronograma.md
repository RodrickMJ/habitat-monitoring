# Cronograma DevOps - Monitoreo de Hábitat para Hámster

## Estructura de Ramas

- `main` → Producción estable
- `qa` → Validación completa
- `dev` → Desarrollo general
- `hu/#` → Historia de Usuario
- `issue/#` → Corrección específica
